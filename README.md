# Commit-X Question 1

## Problem Statement
Given an integer array `arr`, find the contiguous subarray (containing at least one number) which has the largest sum and return its sum.

## Example
### Input
```
9
-2 1 -3 4 -1 2 1 -5 4
```

### Output
```
6
```

### Explanation
The subarray `[4, -1, 2, 1]` has the largest sum of `6`.

### Additional Test Case
#### Input
```
9
1 -2 3 4 -1 2 1 -5 4
```
#### Output
```
9
```
